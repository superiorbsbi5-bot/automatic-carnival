# 📊 Machine Learning: Linear & Logistic Regression

This repository contains implementations of two fundamental supervised machine learning algorithms: **Linear Regression** and **Logistic Regression**. The goal of this project is to understand, apply, and evaluate both regression techniques on real datasets using Python.

---

## 🚀 Project Overview

In this project, I have performed:

- 📈 **Linear Regression** for predicting continuous numerical values  
- 🔍 **Logistic Regression** for binary classification tasks  
- 🧹 Data preprocessing and feature selection  
- 📊 Model training and evaluation  
- 📉 Accuracy and performance analysis  

---

## 🔹 Linear Regression

Linear Regression is used to model the relationship between independent variables and a continuous target variable.

### Key Steps:
- Import dataset
- Handle missing values
- Split data into training and testing sets
- Train Linear Regression model
- Evaluate using metrics like R² score and Mean Squared Error

### Use Case:
- Predicting prices, sales, or other continuous outcomes
